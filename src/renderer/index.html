About AutoFlow

AutoFlow

Video Generator Automation For Flow

Version 1.0.0

AutoFlow is a powerful automation tool for generating videos using Google's Veo 3 and Veo 3 Fast models. It supports both text-to-video and image-to-video generation with batch processing capabilities.

Features:

  • Text-to-Video and Image-to-Video generation
  • Batch processing with concurrent execution
  • Bearer token authentication
  • Real-time progress monitoring
  • Comprehensive logging system
  • Settings persistence
